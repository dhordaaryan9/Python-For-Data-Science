# Python: Data Science & Object-Oriented Programming (OOP)

This repository contains a comprehensive exploration of Python's role in the Data Science ecosystem and the fundamental principles of Object-Oriented Programming (OOP). It serves as a portfolio of practical solutions ranging from basic logic to advanced class-based systems.

---

## 📂 Repository Contents

### 1. 📊 Theoretical Presentation
**File:** `Python’s role in Data Science.pptx`
A professional 7-slide deck covering:
* **Python in Data Science:** An overview of the core stack (NumPy, Pandas, Matplotlib) and why Python acts as the "Glue Language" for the industry.
* **The Four Pillars of OOP:** * **Encapsulation:** Bundling data and methods.
    * **Inheritance:** Code reusability through parent-child classes.
    * **Polymorphism:** Shared interfaces for different classes.
    * **Abstraction:** Simplifying complex internal details.

### 2. 💻 Practical Lab (Jupyter Notebook)
**File:** `python-for-data-science.ipynb`
Contains 20 solved programming challenges organized by complexity:

#### 🔹 Basic Logic & String Engineering
* **Sum of Integers:** Calculating the sum of first $n$ positive integers.
* **Substring Logic:** Pattern matching, frequency analysis, and the "not-poor" replacement logic.
* **Dynamic Formatting:** Smart suffix addition (`ing`/`ly`) and character swapping.

#### 🔹 Data Structures (Lists, Tuples & Dictionaries)
* **Algorithms:** Implementation of GCD and the Fibonacci series.
* **Collection Management:** Unzipping tuples, converting lists to dictionaries, and sorting dictionaries by value.
* **Frequency Mapping:** Counting occurrences of elements using dictionary hashing.

#### 🔹 Advanced Functions
* **Recursion:** Calculating factorials using recursive call stacks.
* **Mathematical Series:** Summing complex odd and even series using factorial denominators.
* **Unique Filtering:** Extracting unique elements while maintaining list integrity.

## 🚀 Mini Project: OOP Password Architect
A class-based implementation of a high-entropy password generator.

* **OOP Implementation:** Features a `User` class that encapsulates profile data (User ID, Name, Password) into an immutable tuple.
* **Security Logic:** Dynamically generates passwords $> 8$ characters by combining user-input seeds with randomized symbols, digits, and uppercase letters.
* **Defensive Coding:** Includes robust **Exception Handling** to manage user input errors gracefully.
